SubPilot
  Stop the subscription leak. Your automated financial co-pilot.
  SubPilot is a privacy-first web application that helps you find and track recurring charges buried in your bank or e-wallet history. Instead of manually typing in every Netflix or Spotify bill, just upload your transaction history, and let SubPilot detect the patterns.

Key Features
  * Smart CSV Import: No need for specific bank integrations. Upload any CSV export (e.g., GCash, Maya, BPI, etc.).
  * Flexible Column Mapping: Map your bank's unique CSV headers (Date, Description, Amount) to SubPilot's engine in seconds.
  * Pattern Detection: Automatically identifies recurring charges to separate subscriptions from one-time splurges.
  * Upcoming Schedule: A clean dashboard that forecasts your next 30 days of bills so you can manage your cash flow proactively.
  * Privacy-Centric: All processing is done in-browser. Your financial data never leaves your machine.

How It Works
   1. Import: Download a CSV export from your bank or e-wallet.
   2. Map: Tell SubPilot which columns contain your transaction data.
   3. Detect: Review the list of recurring charges found by the app.
   4. Monitor: Use the Upcoming tab to see what’s due next.

Getting Started## Prerequisites
  * Node.js (v18 or higher)
  * npm or pnpm

Installation
   1. Clone the repository:
   
   git clone https://github.com
   
   2. Install dependencies:
   
   npm install
   
   3. Run the development server:
   
   npm run dev
   
   4. Open http://localhost:3000 in your browser.
